# Лабораторная работа №2

## Тема
Управление памятью и процессами в ОС Windows

## Цель работы
Изучение процессов Windows и способов управления ими через диспетчер задач и командную строку.

## Выполненные задачи
- запуск процессов;
- просмотр процессов;
- завершение процессов;
- управление explorer.exe;
- использование tasklist и taskkill.

## Используемые команды
- tasklist
- taskkill
- start
- cd
- dir

## Результат
Были изучены механизмы работы процессов Windows и способы управления ими.

# Контрольные вопросы

## 1. Что такое управление вводом-выводом?

Управление вводом-выводом — организация обмена данными между устройствами и оперативной памятью.

---

## 2. Для чего используется диспетчер задач?

Диспетчер задач используется для просмотра и управления процессами, службами и производительностью системы.

---

## 3. Что такое процесс?

Процесс — выполняющаяся программа с выделенными системными ресурсами.

---

## 4. Чем отличается приложение от процесса?

Приложение — программа, а процесс — экземпляр выполняемой программы.

---

## 5. Что такое служба Windows?

Служба — фоновый системный процесс, выполняющий задачи ОС.

---

## 6. Отличие службы от обычного процесса

Службы запускаются автоматически и работают в фоне без участия пользователя.

---

## 7. Основные команды управления процессами

- tasklist
- taskkill
- start
